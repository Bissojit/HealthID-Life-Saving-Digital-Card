# HealthID - Life Saving Digital Card

A lightweight web application that generates an emergency health profile of a person and converts it into a scannable QR code. Designed to provide **critical medical information quickly during emergencies**.

## Features

-  Create an emergency medical profile
-  Stores data securely in browser local storage
-  Generates QR code for instant access
-  Mobile-friendly & printable layout
-  Fast, lightweight and efficient


## Tech Stack

- **Frontend:** HTML, CSS, JavaScript / TypeScript
- **QR Code Generation:** JavaScript QR library
- **Storage:** Browser Local Storage

## Learning Outcomes

- Data persistence using local storage
- QR code generation
- Accessibility-aware form design
- Building practical real-world utilities
- Solving real-world problems
